# ✂️ 剪贴小助手

> 🚀 **开源、离线、超轻量** 的桌面效率工具  
> 🦀 基于 Tauri 2 + Vue 3，体积 < 5 MB，无需联网，随叫随到！

---

## ✨ 功能亮点

### 📜 剪贴板管理
- **文本历史** – 自动记录复制过的文本，一键找回
- **图片历史** – 支持截图/图片剪贴板记录，预览与收藏
- **模糊搜索** – 快速检索历史内容，支持键盘导航
- **窗口置顶** – 一键置顶面板，随时查阅

### ⭐ 收藏夹
- **分组管理** – 创建自定义分组，分类收藏常用内容
- **一键复制** – 双击即可复制收藏内容到剪贴板
- **快速预览** – 悬停预览完整内容

### 📋 待办事项
- **日期切换** – 按日期查看待办，轻松回溯
- **状态管理** – 未开始 / 进行中 / 已完成，全流程跟踪
- **时间冲突检测** – 自动检测时间段冲突，避免安排重叠
- **系统通知提醒** – 待办开始/结束自动推送系统通知，提前提醒

### 🍅 专注时间
- **番茄时钟** – 单次专注或循环专注模式
- **预设时长** – 专注 15/25/30/45/60 分钟快速选择，也支持自定义
- **休息提醒** – 循环模式自动插入休息，支持跳过
- **后台运行** – 专注计时后台持续，Tab 标题实时显示倒计时
- **系统通知** – 专注开始/结束、休息结束自动推送提醒

### 📝 白板
- **临时草稿** – 粘贴到此处快速调整内容，随手记录

### ⚡ 通用特性
- **极速唤醒** – 全局热键一键弹出/隐藏面板
- **完全离线** – 所有数据本地 SQLite 存储，零隐私泄露
- **开机自启** – 可选开机自动启动，开机即用
- **系统托盘** – 最小化到托盘，不占任务栏

## 🎯 默认快捷键

| 热键 | 功能 |
|---|---|
| `Ctrl + Space` | 唤起 / 隐藏面板（全局） |
| `Ctrl + F` | 焦点到搜索框（软件内） |
| `方向键 ↑↓` | 切换剪贴板列表项 |
| `回车` | 搜索框焦点下回车，列表获取焦点 |

## 🛠️ 快速上手

```bash
# 克隆仓库
git clone https://github.com/sallaixu/cut_helper_tauri.git
cd cut_helper_tauri

# 安装依赖
pnpm install          # 也可使用 yarn 或 npm

# 开发运行
pnpm tauri dev

# 构建正式版
pnpm tauri build
```

## 🏗️ 技术栈

| 层级 | 技术 |
|---|---|
| 桌面框架 | Tauri 2 (Rust) |
| 前端框架 | Vue 3 + Composition API |
| UI 组件 | Ant Design Vue |
| 数据存储 | SQLite (tauri-plugin-sql) |
| 系统通知 | tauri-plugin-notification |
| 全局快捷键 | tauri-plugin-global-shortcut |
